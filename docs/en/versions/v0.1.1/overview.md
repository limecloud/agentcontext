# Agent Context v0.1.1

Agent Context v0.1.1 is a patch release that makes Agent Context directly testable against Agent Runtime profile facts without moving runtime ownership into the context standard.

## Highlights

- Adds runtime profile test cases for context envelopes, selection, budgets, assemblies, missing-context records, compaction, redaction, and evidence refs.
- Aligns interoperability guidance with Agent Runtime events, thread read models, task snapshots, and evidence packs.
- Adds optional runtime correlation fields to the public event schema.
- Refreshes README, navigation, LLM entrypoints, release notes, package metadata, and version snapshots for v0.1.1.
