# Agent Context v0.1.0

Agent Context v0.1.0 establishes the first public draft for portable context facts in agent systems.

## Highlights

- Defines context envelopes, surfaces, items, source refs, selection, budgets, windows, assembly, injection, compaction, summaries, missing context, redaction, and events.
- Publishes JSON Schemas for the core objects.
- Adds examples for turn snapshots, memory prefetch, knowledge selection, context compaction, and missing-context requests.
- Adds LLM-friendly entrypoints through `llms.txt`, `llms-full.txt`, `llm.txt`, and `llm-full.txt`.
