# Agent Context v0.1.0

Agent Context v0.1.0 建立首个公共草案，用于表达 Agent 系统中的可移植上下文事实。

## 重点

- 定义 context envelopes、surfaces、items、source refs、selection、budgets、windows、assembly、injection、compaction、summaries、missing context、redaction 与 events。
- 发布核心对象 JSON Schemas。
- 提供 turn snapshot、memory prefetch、knowledge selection、context compaction 与 missing-context request 示例。
- 提供 `llms.txt`、`llms-full.txt`、`llm.txt`、`llm-full.txt`。
