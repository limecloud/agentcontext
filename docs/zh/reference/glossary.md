# 术语表

| 术语 | 含义 |
| --- | --- |
| Context | Agent、模型、工具、UI、runtime、reviewer、user 或 peer agent 在某个时刻可见的信息 |
| Context surface | selection 前可用的候选上下文 |
| Context item | inline 或 referenced 的上下文单元 |
| Source ref | context item 的来源与定位 |
| Selection | candidates、selected、omitted 的决策记录 |
| Budget | token、byte、item、time 或 visibility 限制 |
| Window | 面向目标可见上下文的有序布局 |
| Assembly | 为目标准备的有序 blocks |
| Injection | assembled context 变成目标可见的事件 |
| Compaction | 用 summary 或 digest 替换更大上下文 |
| Missing context | 影响正确性、安全性或完成度的已知缺口 |
| Redaction | 对敏感上下文进行 mask、remove、summarize 或 replace |
